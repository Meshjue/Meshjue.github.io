Coding

For my final project in Art 74, I decided to create a database of information about select character types in the game Super Smash Bros. Ultimate; as a guide for people unacquainted with the fighting game genre. All the footage seen, I recorded and edited in Adobe Premiere. Then, I photoshopped multiple photos for different representations of each character.  I then hand-coded an HTML website using Visual Studio Code to house all this information.

In this project I pulled images together and made an HTML website to create a small narrative of what I believe is one of the most grueling paths an artist can choose. The journey of a Stand-Up comedian. 

Here, I used code in p5js to make a self portrait.
